FMLRHS widget

Get Redhawk sports scores on your site!

Simply copy and paste the following code where you want the widget to appear:

<iframe id="frcontent" src="https://thorlancaster.github.io/FMLRHS/index.html" width="600px" height="200px" style="border: 1px solid black;"></iframe>

You can alter the width and height attributes of the above code to resize the widget, but it looks best if the width is 600px or greater.
