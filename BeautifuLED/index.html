Jinx! - Programming software for BeatifuLED LED signs

System Requirements

  • Java 6 or later (required). Update Java
  • SD card reader or dongle (required to program the sign)
  • 256 MB RAM
  • 16 MB of free disk space
  • Windows XP or greater to run Jinx! If using Linux, wine will kind of work, but it's a pain.
