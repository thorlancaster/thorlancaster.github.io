Since 2014, BeautifuLED LED signs has produced reliable LED signs to show content, attract customers, and help businesses. BeauifuLED LED signs believes that no technical issue is too small and strives to ensure compatiblity and bug-free operation of it's Message Maker programming software. Currently, this first release is fast, easy to use, and free of bugs (at least on Windows). On Linux, no bugs have been observed, but the software has not been tested extensively on Linux.

System Requirements

  • Java 6 or later (required). Update Java
  • SD card reader or dongle (required to program the sign)
  • 256 MB RAM
  • Intel® Pentium or greater (recommended) [anything that runs Java will technically work, like a Raspberry Pi]
  • 16 MB of free disk space

Download software

  • Message maker, version 1.0
  • Jinx by Sven Karshewski
