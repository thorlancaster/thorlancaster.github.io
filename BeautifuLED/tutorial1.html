This LED sign programming tutorial for BMM 1.0 will get your sign up and running in no time at all!

Open this video and follow along


Step 1 - Open BeautifuLED Message Maker

  • Download BMM 1.0 if you haven't already.
  • Unzip the folder to a easily accessible location, like your deskop.
  • Open the folder and double-click on the BMM.jar file. A window should pop up.

Step 2 - Create a frame

  • Click on autosave.bfm in the message hierarchy. autosave.bfm is the file name.
  • In the right side of the window, you will see a button labeled Create empty frame. Click it to create an empty frame.
  • You will notice that a folder has appeared below autosave.bfm, labeled Frame 0. Double click that folder to open frame 0.
  • In the right side of the window, you will see a group of buttons. click add effect 3 times to add 3 effects.

Step 3 - Customize the effects

  • Click on the effect labeled Effect 0: [BLANK] to select it.
  • Using the drop-down menu in the top of the window, change the effect type from [BLANK] to color.
  • Click on the effect labeled Effect 1: [BLANK] to select it.
  • Change the effect type from [BLANK] to text (using the drop down menu you used last time).
  • Change the Foreground Color of the effect to white (Set the red, green, and blue sliders to their max values).
  • Type some text into the box. In the video, it is BeautifuLED LED signs are cool!
  • Increase the text size until it looks good, enable the text outline and set scrolling to Right to Left. Add several (6-10) spaces after the text to ensure that it scrolls most of the way off the sign.
  • Check the Force Integer speed box. The text's speed will be slightly adjusted to sync with the sign's refresh rate. It will look a lot smoother on the LED sign that way. Use antialiasing if it looks better to you that way.
  • Change the position of the text on the sign with the slider labeled position.
  • The text is scrolling by too fast for people to read. Click on Frame 0 on the left and increase the frame's time until the text is moving at a reasonable speed.
  • Click on the effect labeled Effect 2: [BLANK] to select it. Since it is not needed, click Delete this effect to remove it.

Step 4 - Using the mixing snail

  • Create a new frame by clicking on action in the top menu bar, and then clicking Add empty frame.
  • Select the frame and add 3 new effects to it. Double click the frame to expand it.
  • Set the first effect as colored effects > color scroll, the second as text, and the third as colored effects > plasma. Change colors to your liking. Don't worry if it doesn't look right just yet.
  • Click on the frame you just created (on the left), and then click the Use Mixing Snail... button.
  • Set snail control #1 to: Source1:2 Source2:3 Destination:0 with Left intensity as is type.
  • Set snail control #0 to: Source1:0 Source2:1 Destination:0 with left overlay[border] as is type.
  • The text looks kind of faint. Go to the second effect (the text one) and change it's foreground color to white (using Set Foreground Color). You can also change its size and position.
  • More contrast is needed. Select frame 1 and drag the slider on snail control #0 until it looks good.
  • The snail works by applying the snail controls from bottom to top (if they are enabled) sequentially.
  • Don't forget to save using File -> Save!
  • To see more snail-powered mixing, open the example.bfm file by using File -> Open -> Browse to where you downloaded the program -> example.bfm.

Step 5 - Exporting

  • If your message is composed of mainly white backgrounds, that may overheat the sign if the sign is exposed to the sun. Non-white backgrounds are recommended.
  • Ensure the micro SD card is inserted in the adapter and plugged into the computer. The micro SD card should show up on the computer.
  • Select File -> Export to JNR. Under most cases, the program will automatically find the SD card and you can just press save. However, on some systems, you have to browse for the card. It will most likely have a display.jnr file already on it.
  • Once Message Maker starts responding again (not greyed out), you can remove the microSD card from the computer. You don't even have to safely remove, although the computer may complain if you don't.
  • Insert the micro SD card into the LED sign. It should start playing the content right away.

A note on saving

  • When you open Mesage Maker, autosave.bfm will be loaded and you will start where you last left off.
  • To save to other places, use File -> Save.
  • When you close Message Maker, what you are working on will be saved to autosave.bfm.
  • If your computer crashes or suffers a power outage, autosave.bfm will not be saved. Incremental saving is coming in version 2.0.
